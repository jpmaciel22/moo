moo, the DX focused static-site generator.

Olá, bem vindo ao site do moo, seu static-site generator focado em developer experience e liberdade de criação. Para utilizá-lo basta seguir o guia abaixo de instalação e configuração.

Instalação

git clone https://github.com/jpmaciel22/moo.git
npm i
npm link moo

Lógica de pastas

moo/
├── content/ // arquivos yaml
├── templates/ // arquivos .ejs de template
├── layouts/ // layouts do site
├── public/ // pasta que será hosteada ( inclui padrão nativo para hosting no vercel )
└── config.yaml // configurações globais do site ( e.g: navegação )

Exemplo:

# no yaml:
theme: dark
# no template:
data-theme='<%= page.theme %>'
# É SIMPLES ASSIM PARA CRIAR CONTEÚDO DINÂMICO NO MOO.
